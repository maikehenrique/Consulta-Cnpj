## 🧐 About <a name = "about"></a>
### Prerequisites

Make sure you have the following prerequisites installed on your system:
 * Visual Studio 

### Installing and Running <a name = "install"></a>
Follow these steps to set up and run the project on your local environment:

- Clone the repository:
```bash
git clone git@github.com:maikehenrique/Consulta-Cnpj.git
```
- Start the visual studio and compile the project with command F5

## 🔧 Running <a name = "running"></a>
- To use the query, just inform the cpnj and press the query button
